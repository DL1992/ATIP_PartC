This awesome game is a maze simulator for everyone!
boy or girl, 3 or 93. everyone can enjoy playing this.

So you must be asking yourself "how this masterpiece came to be?".

the maze itself is generated by a very complex and cool algorithm!
there are a lot of ways to create a random maze.
for this game we chose to use the prim algorithm
which you can learn more about it in the link below. (it's really fun!).

to solve the maze and give you the very best and short solution
we used several different search algorithms - BFS, DFS, Best first search.
all of which you can read more in the links below.

the whole game supprot the mvvm architecture design. combining 3 layers into
one glorious game.
